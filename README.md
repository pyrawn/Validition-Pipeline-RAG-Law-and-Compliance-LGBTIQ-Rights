# Validition-Pipeline-RAG-Law-and-Compliance-LGBTIQ-Rights
RAG-based NLP pipeline for automated legal gap analysis of Mexican state laws on LGBTIQ+ rights. Evaluates compliance against legal markers (L1–L3, C1–C3) using LlamaIndex, returning structured scores, justifications, and cited articles per state. Part of the Mexico Equality &amp; Cyber Literacy Atlas.
